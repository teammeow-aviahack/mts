# Создание виртуального окружения

1. Прописать команду 

```python -m venv venv```

2. Активировать виртуальное окружения

Win:

```venv/Scripts/activate.bat```

MacOS/Linux:

```source venv/bin/activate```

3. Установить необходимые пакеты 

```pip install -r requierements.txt```
